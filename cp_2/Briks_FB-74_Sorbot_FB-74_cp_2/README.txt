Creator.py

В данный фал мы записываем наш ключ, который передается в Crypt.py и Decrypt.py, которые данный файл создает
Вывод в файл Key.txt


Crypt.py

Шифрует сообщение с заданным ключом
Вывод в файл Crypted_text.txt

Decrypt.py

Дешифрует сообщение тем же ключом
Вывод в файл Decrypted_text.txt

IS.py

Подсчет частоты букв в тексте и ИС

Analis.py

Частотный анализ для нахождения ключа в задании варианта
